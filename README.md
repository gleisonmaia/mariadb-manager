# MariaDB Log Explorer

Aplicação desktop (WPF, .NET Framework 4.8) para ler o binary log do MariaDB, persistir eventos e consultar o histórico.

## Documentação

- [Instruções de compilação e uso](docs/Instrucoes.md)
- [Requisitos do projeto](docs/Requisitos.md)

## Repositório remoto (opcional)

O script `repo-remoto.ps1` na raiz ajuda a criar o repositório no GitHub e enviar a branch `main` (requer [GitHub CLI](https://cli.github.com/) autenticado).
